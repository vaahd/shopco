# SHOP.CO - Modern E-Commerce Platform

A premium, fully responsive e-commerce project built with React, Vite, and Tailwind CSS.

## Features

-   **Clean Architecture**: Professional folder structure (API, Services, Layouts).
-   **Safe Forms**: Robust validation with React Hook Form and Zod.
-   **Rich Experience**: Skeleton loaders, animated page transitions, and SEO optimization.
-   **Dark Mode**: Native implementation with `next-themes`.
-   **Shadcn UI**: High-end UI components and dynamic layout.
-   **Fully Responsive**: Built for mobile-first experience.

## Tech Stack

-   **Framework**: React 18
-   **Styling**: Tailwind CSS
-   **Tooling**: Vite, TypeScript
-   **UI Components**: Shadcn-UI, Lucide-React
-   **State Management**: React Query, Context API

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Start the development server: `npm run dev`.
4.  Build for production: `npm run build`.

## Project Structure

Refer to `src/types/index.ts` for core data models and `src/services/` for business logic and API integration.
